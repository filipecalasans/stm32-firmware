8.284 UMASKL — Unsigned left justified mask

Synopsis:

RESULT = UMASKL(I[, KIND])

Description:

UMASKL(I[, KIND]) has its leftmost I bits set to 1, and the remaining bits set to 0.

Class:

Elemental function

Arguments:
IShall be of type UNSIGNED.
KINDShall be a scalar constant expression of type INTEGER.
Return value:

The return value is of type UNSIGNED. If KIND is present, it specifies the kind value of the return type; otherwise, it is of the default unsigned kind.

Standard:

Extension (see Unsigned integers)
